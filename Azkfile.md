## Start azk

#### [Install azk](http://docs.azk.io/en/installation/README.html)

```sh
# easy install
curl -Ls http://azk.io/install.sh | bash
```

#### start all systems

```sh
# easy start
azk start
```

#### deploy to Digital Ocean

```sh
# commit your changes, and:

azk shell deploy  # needed the first-time

# OR
azk shell fast-deploy
```

--------------------

### Other azk commands

#### stop all containers

```sh
$ azk stop
```

#### restart all container

```sh
$ azk restart
```

#### restart and reprovision all container

```sh
$ azk restart -Rvv
```

#### check logs

```sh
$ azk logs
```

#### info on containers

```sh
$ azk info
```
